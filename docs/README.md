# 操作系统引论

> 计算机系统由 **硬件** 和 **软件** 组成, 操作系统是配置机硬件上的第一层软件, 是对硬件得当首次扩充.
> 操作系统已成为现代计算机系统(大, 中, 小, 及微型机), 多处理机系统, 计算机网络, 多媒体系统以及嵌入式系统中都义序配置的, 最重要的系统软件

## 操作系统的目标和作用

### 目标
1. 有效性
2. 方便性
3. 可扩充性
4. 开放性

### 作用
1. OS作为用户与计算机硬件系统之间的接口
2. OS作为计算机系统资源的管理者
3. OS实现了对计算机资源的抽象
